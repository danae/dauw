Dauw documentation
==================

Dauw is a high-level, statically typed, object-oriented, general-purpose scripting language, taking inspiration from `Lua <https://en.wikipedia.org/wiki/Lua_(programming_language)>`_, `Nim <https://en.wikipedia.org/wiki/Nim_(programming_language)>`_ and `Python <https://en.wikipedia.org/wiki/Python_(programming_language)>`_.
